#### flask 세팅하기

- window + R 단축키 cmd 명령어 실행
```
cd C:\~\projects\all_about_jobs 
```
- 아래 명령어를 입력 <br>
```
python -m venv env
```
- 가상의 환경 적용 시키는 스크립트 실행 <br>
```
env\Scripts\activate.bat
```
- vscode 실행 <br>
```
. code
```
- vscode 터미널(단축키 ctrl + `) 다음 명령어를 입력해서 flask 설치 <br>
```
pip install flask
```
- app.py 파일 생성 후 터미널에서 다음 명령어 실행
```
flask run
```
